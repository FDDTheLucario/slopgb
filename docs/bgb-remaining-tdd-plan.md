# bgb-clone — remaining-tasks TDD plan

Generated by `/tdd-test-plan`. Finishes the functional-1:1 bgb clone: the debugger
Execution-profiler / Search menus, Evaluate-expression, the game-window Load-ROM /
Recent-ROMs path flow, asm export — plus a prerequisite god-file split (the frontend
`main.rs`/`debugger.rs` grew past the 1000-line cap this session). Every menu item is
transcribed from a REAL capture in [`docs/bgb-reference/menus/`](bgb-reference/menus/),
never invented. Frontend = winit/softbuffer/cpal only (no new deps); core = std-only,
no unsafe, golden-safe `&self`/optional-`&mut` debug surface.
