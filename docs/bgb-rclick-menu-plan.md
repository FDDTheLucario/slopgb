# bgb right-click-menu + menus TDD plan

Generated by `/tdd-test-plan`. Functional-1:1 clone of **bgb's context (right-click) menus
and menu-bar dropdowns across every window** — main LCD, debugger, VRAM viewer, I/O map — on
slopgb's own software UI (`ui::` toolkit + the M1 mouse-routing infra; winit/softbuffer only,
no GUI deps).

**Every menu item below is transcribed from a REAL captured bgb screenshot** in
[`docs/bgb-reference/menus/`](bgb-reference/menus/) — never invented. Captures so far (bgb 1.6.4,
Pokemon Crystal, wine, this machine):
